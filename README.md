# Página de Vendas — Como Controlar a Ansiedade em 30 Segundos

## Passos rápidos
1. Substitua `REPLACE_CHECKOUT_URL` no index.html pelo link do checkout.
2. Substitua `https://SEU-USUARIO.github.io/SEU-REPO/` por sua URL real.
3. Adicione as imagens em `assets/images/` (nomes exatos listados abaixo).
4. Faça push para o repositório e ative GitHub Pages (branch main / root).

## Arquivos de imagem essenciais
- assets/images/thiago-hero.jpg (1600x900)
- assets/images/thiago-portrait.jpg (800x800)
- assets/images/thiago-action.jpg (1200x800)
- assets/images/og-image.jpg (1200x630)
- assets/images/favicon-32.png (32x32)

## Observações
- Troque depoimentos fictícios por reais antes de tráfego pago.
- Se quiser imagens em WEBP, basta substituir mantendo o mesmo nome de arquivo.
